To-Do App
A simple and interactive To-Do tracking application built with HTML, CSS, and JavaScript.

Features
Add Tasks: Quickly add new tasks to your list.
Mark as Completed: Check off tasks as you finish them.
Delete Tasks: Remove individual tasks you no longer need.
Clear Completed: Easily remove all completed tasks at once.
Task Counter: Keep track of how many tasks you still have remaining.
Persistent Storage: Your tasks are saved in your browser's local storage, so they'll still be there when you come back.
Technologies Used
HTML5
CSS3
JavaScript (ES6+)
How to Run Locally
Clone this repository.
Open index.html in your web browser.
Enjoy managing your tasks!
